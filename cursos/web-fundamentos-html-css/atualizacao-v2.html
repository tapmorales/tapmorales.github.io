Atualização Curso Web Design Fundamentos

Última atualização: 14/06/2023

Histórico de atualização

Abaixo você confere as seções em ambas as versões e a relação entre elas.

Por exemplo, na versão 1, a "Seção 1: Introdução" foi totalmente revista e regravada. Por esse motivo o novo conteúdo foi adicionado e o conteúdo antigo foi renomeado para "Seção 15: [LEGADO] Introdução" na versão atualizada.

Clique sobre cada uma das seções abaixo para obter maiores detalhes.
